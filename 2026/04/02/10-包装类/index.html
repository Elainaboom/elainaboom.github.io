包装类& 数学类

1 包装类

把基本数据类型包装成引用数据类型

byte short int long float double char boolean void
Byte Short Integer Long Float Double Character Boolean Void

包装类型的数值类型 Byte、Short、Integer、Long、Float、Double。它们几个有共同的父类Number

自动装箱底层原理:底层调用的是valueOf(Xxx)

Byte在-128到127的范围内返回同一个对象

Short、Integer、Long在-128到127的范围内返回同一个对象，超出范围返回新的对象

Float和Double永远是创建新的对象

Character在0-127的范围内返回同一个对象,超出范围返回新的对象

Boolean返回TRUE或者FALSE
