数据类型

1. Java 8种基本数据类型（核心）

类型分类 关键字 占用字节 取值范围 默认值 关键注意事项
整数型 byte 1 -128 ~ 127（$2^7-1$） 0 节省内存，常用于文件/网络传输
整数型 short 2 -32768 ~ 32767（$2^{15}-1$） 0 日常使用少，偶尔用于数组索引
整数型 int 4 -2147483648 ~ 2147483647（$2^{31}-1$） 0 整数默认类型，日常计算首选
整数型 long 8 $-2^{63}$ ~ $2^{63}-1$ 0L 声明需加L（大写），用于大整数（时间戳）
浮点型 float 4 ±3.4e38（精度6-7位） 0.0f 声明需加f，用于低精度小数（游戏坐标）
浮点型 double 8 ±1.8e308（精度15-17位） 0.0d 小数默认类型，日常小数计算首选
字符型 char 2 ‘\u0000’（0）~ ‘\uffff’（65535） ‘\u0000’ 支持中文（Unicode编码），单引号包裹
布尔型 boolean 1/4 true / false（无数值范围） false 不能用0/1替代，仅用于条件判断

2. 基本数据类型 vs 引用数据类型（核心区别）

对比维度 基本数据类型 引用数据类型
存储内容 直接存储值 存储对象的内存地址（引用）
内存位置 栈内存 引用在栈，对象在堆
默认值 如int=0、boolean=false null（空引用）
内存大小 固定（如int=4字节） 不固定（由对象大小决定）
比较方式 == 比较值是否相等 == 比较地址，equals() 比较内容
典型代表 int、double、boolean String、数组、自定义类、接口

3. 基本类型 ↔ 包装类对应关系（自动装箱/拆箱）

基本数据类型 包装类（引用类型） 自动装箱示例 自动拆箱示例 常用工具方法
byte Byte Byte b = 10; byte num = b; Byte.parseByte(“10”)
short Short Short s = 100; short num = s; Short.parseShort(“100”)
int Integer Integer i = 1000; int num = i; Integer.parseInt(“1000”)
long Long Long l = 10000L; long num = l; Long.parseLong(“10000”)
float Float Float f = 3.14f; float num = f; Float.parseFloat(“3.14”)
double Double Double d = 3.1415; double num = d; Double.parseDouble(“3.1415”)
char Character Character c = ‘a’; char ch = c; Character.isDigit(‘9’)
boolean Boolean Boolean flag = true; boolean b = flag; Boolean.parseBoolean(“true”)
