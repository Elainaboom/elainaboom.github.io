2 Java中的内存分配

Java当中的内存区域划分为5块

  • 栈内存:存储的都是局部变量
    • 凡是定义在方法的大括号的变量，都是局部变量
    • 局部变量一旦超出自己的作用域(所属的大括号)，就不能再使用，内存空间会被回收。
    • 所有的代码的执行都在栈里面
  • 堆内存:存储的都是对象(凡是new出来的东西，都在堆内存中)
    • 对象存储在堆内存中，具有一个首地址值，其中不同的数据类型，都会被赋予一个初始的默认值
    • 基本数据类型的默认值
      • byte/short/int 默认值是 0
      • long 默认值是0L
      • float 默认值是0.0F
      • double 默认值是0.0
      • char 默认值是 ‘\u0000’
      • boolean 默认值是 false
    • 引用数据类型的默认值 null
      • null可以赋值给引用数据类型
  • 元空间: 存储与class相关的信息
  • 本地方法栈: 和操作系统相关
  • 寄存器(PC计数器): 和CPU相关
